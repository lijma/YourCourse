# WeTalk Api
> 提供wetalk api 数据服务

## Tech Stack
* oracle java 17
* springboot 3.0.0
* postgres 12
* gradle 7.5.1
* springcloud 2022.0.0-RC2
* auth0 4.2.1
* lombok
* mybatis
* mybatis-plus

## 参与研发
* clone master
* create branch and checkout
* do code changes and push to branch
* create pr merge to master
* ask owner to review

## 本地开发
```shell
./gradlew clean build
ACTIVE_PROFILE=local ./gradlew bootRun
```
